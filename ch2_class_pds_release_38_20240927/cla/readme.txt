Chandrayaan-2 Large Area Soft x-ray Spectrometer (CLASS) Data Archive
Chandrayaan-2 Mission to Moon

This bundle (cla) contains the following collections (directories) :
--calibration
--data (to be separately downloaded and organised in this directory)
--document
--miscellaneous

It also contains the following files :
--bundle_cla.xml
--readme.txt (this file)


Every directory contains the relevant files along with their label files in 
XML format and a corresponding inventory file in CSV format and its label 
file in XML format.

Brief description of the collection directories
-----------------------------------------------

calibration collection
- contains the various versions of the calibration files to be used along 
with the data

data collection
- contains both the calibrated (L1) and derived data (L2) organised as 
'calibrated' directory having data in the YYYY/MM/DD subdirectory format and 
'derived' directory having data in subdirectory for every element.

document collection
- contains the Software Interface Specification (SIS) and User Manual 
documents

miscellaneous collection
- contains the analysis software (as a zip file)

The CLASS Software Interface Specification (SIS) is the primary documentation
for the CLASS archive bundle.

Detailed descriptions of the file naming conventions of every data product 
are given in SIS document available under 'document' collection

This bundle is created by the CLASS Data Processing Team at Space Astronomy 
Group (SAG), URSC, ISRO.
Archival is done by ISDA, ISSDC, ISTRAC, ISRO
